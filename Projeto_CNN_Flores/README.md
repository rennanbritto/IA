# 🌻 Flower Classification using MobileNetV2

Projeto de classificação de imagens de flores utilizando Deep Learning e Transfer Learning com MobileNetV2.

## Dataset

Dataset disponível em:

[https://www.kaggle.com/datasets/alxmamaev/flowers-recognition](https://www.kaggle.com/datasets/rahmasleam/flowers-dataset)

Flower Photos Dataset contendo:

- Daisy
- Dandelion
- Roses
- Sunflowers
- Tulips

## Arquitetura

- MobileNetV2 (pré-treinada em ImageNet)
- Global Average Pooling
- Dense Layer
- Softmax (5 classes)

## Resultados

- Accuracy > 85%
- Precision > 85%
- Recall > 85%
- F1-Score > 85%
- AUC entre 0.96 e 0.99
